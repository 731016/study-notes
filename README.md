### git上传代码

  [简单解决 gitee 上传限制问题 - jaychou、 - 博客园 (cnblogs.com)](https://www.cnblogs.com/jaychou-/p/14983818.html#:~:text=我们使用代码来上,10m以内的文件)

```Git
## 克隆仓库 
git clone 仓库地址

## cd进入克隆的仓库
cd 进入文件目录
把添加的东西放入 仓库内

## 添加到缓存区
git add .

## 提交到本地仓库
git commit -m "注释"

## 上传到远程仓库
git push origin master
```
